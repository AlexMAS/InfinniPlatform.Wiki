Элемент для отображения печатного представления.

# Extends

[`Element`](../../KeyConcepts/Element/)

# Syntax

```js
new DocumentViewer([parent])
```

## Parameters

`parent`

Необязательный. Родительский [элемент](../../KeyConcepts/Element/).
