Устанавливает символ маскировки символов пароля.

# Syntax

```js
PasswordChar.setPasswordChar(value)
```

## Parameters

`value`

Cимвол маскировки символов пароля.

# Examples

```js
PasswordChar.setPasswordChar('*');
```
