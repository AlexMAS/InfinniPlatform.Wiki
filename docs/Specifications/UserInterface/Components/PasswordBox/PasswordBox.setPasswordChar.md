Устанавливает символ маскировки символов пароля.

# Syntax

```js
PasswordBox.setPasswordChar(value)
```

## Parameters

`value`

Cимвол маскировки символов пароля.

# Examples

```js
PasswordBox.setPasswordChar('*');
```
