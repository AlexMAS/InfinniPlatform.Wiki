День недели.

## Syntax

```js
DayOfWeek = {
  sunday: 0,
  monday: 1,
  tuesday: 2,
  wednesday: 3,
  thursday: 4,
  friday: 5,
  saturday: 6
}
```

### Members

`sunday`

Воскресенье.

`monday`

Понедельник.

`tuesday`

Вторник.

`wednesday`

Среда.

`thursday`

Четверг.

`friday`

Пятница.

`saturday`

Суббота.

```js
{
```
