Устанавливает [цвет](../../Style/ColorStyle/) текста элемента.

# Syntax

```js
Element.setForeground(value)
```

## Parameters

`value`

[Цвет](../../Style/ColorStyle/) текста элемента.

# Examples

```js
Element.setForeground(ColorStyle.primary1);
```
