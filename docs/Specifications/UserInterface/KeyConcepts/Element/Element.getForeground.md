Возвращает [цвет](../../Style/ColorStyle/) текста элемента.

# Syntax

```js
Element.getForeground()
```

## Returns

[Цвет](../../Style/ColorStyle/) текста элемента.

# Examples

```js
var foreground = Element.getForeground();
```
