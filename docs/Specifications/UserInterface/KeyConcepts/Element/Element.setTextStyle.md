Устанавливает [стиль текста](../../Style/TextStyle/) элемента.

# Syntax

```js
Element.setTextStyle(value)
```

## Parameters

`value`

[Стиль текста](../../Style/TextStyle/) элемента.

# Examples

```js
Element.setTextStyle(TextStyle.body1);
```
