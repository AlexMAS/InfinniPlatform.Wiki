Кнопка мыши.

# Syntax

```js
MouseButton = {
  left: 1,
  right: 2,
  wheel: 4
}
```

## Members

`left`

Левая кнопка мыши.

`right`

Правая кнопка мыши.

`wheel`

Колесо или средняя кнопка мыши.
