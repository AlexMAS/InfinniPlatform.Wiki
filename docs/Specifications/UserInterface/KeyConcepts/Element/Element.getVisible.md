Возвращает значение, определяющее, отображается ли элемент в интерфейсе.

# Syntax

```js
Element.getVisible()
```

## Returns

Логическое значение, определяющее, отображается ли элемент в интерфейсе.

# Examples

```js
var elementVisible = Element.getVisible();
```
