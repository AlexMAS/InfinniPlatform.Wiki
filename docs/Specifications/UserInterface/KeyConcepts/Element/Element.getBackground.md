Возвращает [цвет](../../Style/ColorStyle/) фона элемента.

# Syntax

```js
Element.getBackground()
```

## Returns

[Цвет](../../Style/ColorStyle/) фона элемента.

# Examples

```js
var background = Element.getBackground();
```
