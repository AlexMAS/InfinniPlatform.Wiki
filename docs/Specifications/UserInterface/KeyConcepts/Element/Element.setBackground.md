Устанавливает [цвет](../../Style/ColorStyle/) фона элемента.

# Syntax

```js
Element.setBackground(value)
```

## Parameters

`value`

[Цвет](../../Style/ColorStyle/) фона элемента.

# Examples

```js
Element.setBackground(ColorStyle.primary1);
```
