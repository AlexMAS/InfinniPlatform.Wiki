Возвращает [горизонтальное выравнивание в родительском элементе](../ElementHorizontalAlignment/).

## Syntax

```js
Element.getHorizontalAlignment()
```

## Examples

```js
var horizontalAlignment = Element.getHorizontalAlignment();
```
