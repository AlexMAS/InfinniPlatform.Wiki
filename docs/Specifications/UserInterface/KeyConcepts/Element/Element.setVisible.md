Устанавливает значение, определяющее, отображается ли элемент в интерфейсе.

# Syntax

```js
Element.setVisible(value)
```

## Parameters

`value`

Логическое значение, определяющее, отображается ли элемент в интерфейсе.

# Examples

```js
Element.setVisible(true);
```
