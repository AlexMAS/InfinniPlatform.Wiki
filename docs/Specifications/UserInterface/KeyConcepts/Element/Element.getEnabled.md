Возвращает значение, определяющее, возможен ли доступ к элементу.

# Syntax

```js
Element.getEnabled()
```
## Returns

Логическое значение, определяющее, возможен ли доступ к элементу.

# Examples

```js
var elementEnabled = Element.getEnabled();
```
