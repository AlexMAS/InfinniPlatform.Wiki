Устанавливает значение, определяющее, возможен ли доступ к элементу.

# Syntax

```js
Element.setEnabled(value)
```

## Parameters

`value`

Значение, определяющее, возможен ли доступ к элементу.

# Examples

```js
Element.setEnabled(true);
```
