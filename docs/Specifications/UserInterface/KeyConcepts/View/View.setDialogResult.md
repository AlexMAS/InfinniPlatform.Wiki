Устанавливает [результат работы представления](../DialogResult/).

# Syntax

```js
View.setDialogResult(value)
```

## Parameters

`value`

[Результат работы представления](../DialogResult/).

# Examples

```js
View.setDialogResult(DialogResult.accepted);
```
