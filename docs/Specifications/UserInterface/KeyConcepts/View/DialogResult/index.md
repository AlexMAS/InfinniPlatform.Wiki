Результат работы диалогового окна.

# Syntax

```js
DialogResult = {
  none: 0,
  accepted: 1,
  canceled: 2
}
```

## Members

`none`

Не определен.

`accepted`

Пользователь подтвердил диалог.

`canceled`

Пользователь отменил диалог.
