Удаляет [скрипт](../../Script/) из представления.

# Syntax

```js
View.removeScript(name)
```

## Parameters

`name`

Наименование скрипта. Должно быть уникальным в рамках представления.

# Examples

```js
View.removeScript('myAddFunction')
```
