Возвращает [контейнер элементов представления](../../../LayoutPanels/).

# Syntax

```js
View.getLayoutPanel()
```

## Returns

[Контейнер элементов представления](../../../LayoutPanels/).

# Examples

```js
var layoutPanel = View.getLayoutPanel();
```
