Удаляет [параметр](../../Parameter/) из представления.

# Syntax

```js
View.removeParameter(name)
```

## Parameters

`name`

Наименование параметра. Должно быть уникальным в рамках представления.

# Examples

```js
View.removeParameter('myParameter');
```
