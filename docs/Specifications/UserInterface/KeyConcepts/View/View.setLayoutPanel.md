Устанавливает [контейнер элементов представления](../../../LayoutPanels/).

# Syntax

```js
View.setLayoutPanel(value)
```

## Parameters

`value`

[Контейнер элементов представления](../../../LayoutPanels/).

# Examples

```js
View.setLayoutPanel(new GridPanel());
```
