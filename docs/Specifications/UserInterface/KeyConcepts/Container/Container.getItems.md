Возвращает [коллекцию](../../Collection/) дочерних элементов контейнера.

# Syntax

```js
Container.getItems()
```

## Returns

[Коллекция](../../Collection/) дочерних элементов контейнера.

# Examples

```js
var items = Container.getItems();
```
