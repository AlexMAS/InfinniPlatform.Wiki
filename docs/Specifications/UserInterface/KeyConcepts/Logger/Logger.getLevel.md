Возвращает минимальный [уровень регистрируемых сообщений](../LogLevel/).

# Syntax

```js
Logger.getLevel()
```

## Returns

Минимальный [уровень регистрируемых сообщений](../LogLevel/).

# Examples

```js
var level = Logger.getLevel();
```
