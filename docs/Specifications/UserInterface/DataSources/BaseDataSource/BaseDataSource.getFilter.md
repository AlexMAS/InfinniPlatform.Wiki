Возвращает фильтр элементов.

# Description

Фильтр элементов представляется массивом [критериев фильтрации](../Criteria/).

# Syntax

```js
BaseDataSource.getFilter()
```

## Returns

Массив [критериев фильтрации](../Criteria/).

# Examples

```js
var filter = BaseDataSource.getFilter();
```
