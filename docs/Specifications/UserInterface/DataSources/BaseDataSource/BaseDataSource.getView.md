Возвращает родительское [представление](../../../KeyConcepts/View/).

# Syntax

```js
BaseDataSource.getView()
```

## Returns

Родительское [представление](../../../KeyConcepts/View/) источника данных.

# Examples

```js
var parentView = BaseDataSource.getView();
```
