position: 21
---

Разрешает обновление списка элементов.

# Syntax

```js
BaseDataSource.resumeUpdate()
```

# Examples

```js
BaseDataSource.suspendUpdate();
BaseDataSource.setPageNumber(5);
BaseDataSource.setPageSize(20);
BaseDataSource.resumeUpdate();
```
