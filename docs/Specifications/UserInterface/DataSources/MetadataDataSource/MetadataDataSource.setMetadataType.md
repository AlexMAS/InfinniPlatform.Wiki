Устанавливает [тип метаданных](../MetadataType/).

# Syntax

```js
MetadataDataSource.setMetadataType(value)
```

## Parameters

`value`

[Тип метаданных](../MetadataType/).

# Examples

```js
MetadataDataSource.setMetadataType(MetadataType.view);
```
