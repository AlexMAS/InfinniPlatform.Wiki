Устанавливает дочерний элемент ячейки.

# Syntax

```js
GridCell.setItem(value)
```

## Parameters

`value`

[Визуальный элемент](../../../../KeyConcepts/Element/), который необходимо расположить внутри ячейки.

# Examples

```js
var item = new Label(view);
GridCell.setItem(item);
```
