Возвращает дочерний элемент ячейки.

# Syntax

```js
GridCell.getItem()
```

## Returns

[Визуальный элемент](../../../../KeyConcepts/Element/), расположенный внутри ячейки.

# Examples

```js
var item = GridCell.getItem();
```
