Возвращает [контейнер элементов](../../../) страницы.

# Syntax

```js
TabPage.getLayoutPanel()
```

## Returns

[Контейнер элементов](../../../) страницы.

# Examples

```js
var layoutPanel = TabPage.getLayoutPanel();
```
