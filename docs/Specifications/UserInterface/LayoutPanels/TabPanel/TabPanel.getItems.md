Возвращает [коллекцию](../../../KeyConcepts/Collection/) дочерних элементов панели.

# Syntax

```js
TabPanel.getItems()
```

## Returns

[Коллекция](../../../KeyConcepts/Collection/) дочерних элементов панели.

# Examples

```js
var items = TabPanel.getItems();
```
