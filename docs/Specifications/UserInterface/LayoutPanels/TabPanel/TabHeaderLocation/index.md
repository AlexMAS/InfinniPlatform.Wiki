Расположение закладок в [`TabPanel`](../).

# Syntax

```js
TabHeaderLocation = {
  none: 'None',
  left: 'Left',
  top: 'Top',
  right: 'Right',
  bottom: 'Bottom',
}
```

## Members

`none`

Закладки не отображаются.

`left`

Закладки отображаются слева.

`top`

Закладки отображаются сверху.

`right`

Закладки отображаются справа.

`bottom`

Закладки отображаются снизу.
