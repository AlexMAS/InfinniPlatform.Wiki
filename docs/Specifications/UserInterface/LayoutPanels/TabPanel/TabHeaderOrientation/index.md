Расположение закладок в [`TabPanel`](../).

# Syntax

```js
TabHeaderOrientation = {
  horizontal: 'Horizontal',
  vertical: 'Vertical'
}
```

## Members

`horizontal`

Текст закладок отображается по горизонтали.

`vertical`

Текст закладок отображается по вертикали.
