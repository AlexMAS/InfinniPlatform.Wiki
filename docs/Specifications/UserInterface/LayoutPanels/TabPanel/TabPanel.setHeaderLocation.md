Устанавливает [расположение закладок](../TabHeaderLocation/).

# Syntax

```js
TabPanel.setHeaderLocation(value)
```

## Parameters

`value`

[Расположение закладок](../TabHeaderLocation/).

# Examples

```js
TabPanel.setHeaderLocation(TabHeaderLocation.top);
```
