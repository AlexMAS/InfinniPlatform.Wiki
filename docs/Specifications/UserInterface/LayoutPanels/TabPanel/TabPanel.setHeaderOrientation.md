Возвращает [ориентацию закладок](../TabHeaderOrientation/).

# Syntax

```js
TabPanel.setHeaderOrientation(value)
```

## Parameters

`value`

[Ориентация закладок](../TabHeaderOrientation/).

# Examples

```js
TabPanel.setHeaderOrientation(TabHeaderOrientation.horizontal);
```
