Устанавливает [ориентацию стека элементов](../StackPanelOrientation/).

# Syntax

```js
StackPanel.setOrientation(value)
```

## Parameters

`value`

[Ориентация стека элементов](../StackPanelOrientation/).

# Examples

```js
StackPanel.setOrientation(StackPanelOrientation.horizontal);
```
