Ориентация [`StackPanel`](../).

# Syntax

```js
StackPanelOrientation = {
  horizontal: 'Horizontal',
  vertical: 'Vertical'
}
```

## Members

`horizontal`

Стек ориентирован горизонтально.

`vertical`

Стек ориентирован вертикально.
