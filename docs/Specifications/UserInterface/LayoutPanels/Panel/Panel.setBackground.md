Устанавливает [цвет](../../../KeyConcepts/Style/ColorStyle/) фона панели.

# Syntax

```js
Panel.setBackground(value)
```

## Parameters

`value`

[Цвет](../../../KeyConcepts/Style/ColorStyle/) фона панели.

# Examples

```js
Panel.setBackground(ColorStyle.primary1);
```
