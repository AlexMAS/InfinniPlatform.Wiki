Возвращает [цвет](../../../KeyConcepts/Style/ColorStyle/) фона панели.

# Syntax

```js
Panel.getBackground()
```

## Returns

[Цвет](../../../KeyConcepts/Style/ColorStyle/) фона панели.

# Examples

```js
var background = Panel.getBackground();
```
