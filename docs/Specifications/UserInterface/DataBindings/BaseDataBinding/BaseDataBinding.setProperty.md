Устанавливает путь к свойству объекта источника данных.

# Syntax

```js
BaseDataBinding.setProperty(value)
```

## Parameters

`value`

Путь к свойству объекта источника данных.

# Examples

```js
BaseDataBinding.setProperty('Property1');
```
