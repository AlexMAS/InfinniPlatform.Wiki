Устанавливает текст для отображения ложного значения.

# Syntax

```js
Format.setFalseText(value);
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`value`|`String`|Текст для отображения ложного значения.|

# Returns

Нет.

# Examples

```js
format.setFalseText('Нет');
```
