Возвращает фильтр по идентификатору элемента.

# Syntax

```js
DataProvider.createIdFilter(itemId)
```

## Parameters

|Name|Description|
|----|-----------|
|itemId|Идентификатор элемента|

## Returns

Фильтр, соответсвующий идентификатору itemId.

# Examples

```js
var idFilter = dataProvider.createIdFilter(itemId);
```
