Возвращает [стиль текста](/docs/API/Core/Style/TextStyle/) элемента.

# Syntax

```js
Element.getBackground()
```


## Parameters

Нет

## Returns

[Стиль текста](/docs/API/Core/Style/TextStyle/) элемента.

# Examples

```js
var textStyle = Element.getTextStyle();
```
