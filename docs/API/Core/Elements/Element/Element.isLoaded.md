Проверяет, готов ли элемент. Возвращает соответствующее значение.

# Syntax

```js
Element.isLoaded()
```

## Parameters

Нет

## Returns

Возвращает `true`, если элемент готов, иначе возвращает `false`.

# Examples

```js
var isLoaded = Element.isLoaded();
```
