Возвращает всплывающую подсказку элемента.

# Syntax

```js
Element.getToolTip()
```

## Parameters

Нет

## Returns

Текст или визуальный элемент, появляющийся при наведении на элемент.

# Examples

```js
var toolTip = Element.getToolTip();
```
