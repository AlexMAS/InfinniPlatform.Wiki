Устанавливает текстуру элемента.

# Syntax

```js
Element.setTexture(value)
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|value|`String`|Текстура элемента.|

## Returns

Метод не возвращает значений.

# Examples

```js
Element.setTexture('grid');
```
