Возвращает [цвет](/docs/API/Core/Style/ColorStyle/) фона элемента.

# Syntax

```js
Element.getBackground()
```

## Parameters

Нет

## Returns

[Цвет](/docs/API/Core/Style/ColorStyle/) фона элемента.

# Examples

```js
var background = Element.getBackground();
```
