Возвращает кастомный стиль элемента (на деле это класс, стили которого
могут быть заданы в подключаемых на страницу стилях).

# Syntax

```js
Element.getStyle()
```

## Parameters

Нет

## Returns

Стилевой класс элемента. 

# Examples

```js
var styleName = Element.getStyle();
```
