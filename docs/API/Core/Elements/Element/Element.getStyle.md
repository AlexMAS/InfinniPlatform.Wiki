Возвращает кастомный стиль элемента (на деле это класс, стили которых 
могут быть заданы в подключаемых на страницу стилях).

# Syntax

```js
Element.getStyle()
```

## Returns

Стилевой класс элемента. 

# Examples

```js
var texture = Element.getStyle();
```
