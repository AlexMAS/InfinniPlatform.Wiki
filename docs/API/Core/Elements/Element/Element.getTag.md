Возвращает специальное значение, хранящяеся в элементе. Тег можно использовать для привязки произвольных значений к элементу.

# Syntax

```js
Element.getTag()
```

## Parameters

Нет

## Returns

Значение хранящяеся в элементе.

# Examples

```js
var value = Element.getTag();
```
