Устанавливает [режим отображения представления](../OpenMode/).

# Syntax

```js
linkView.setOpenMode(value)
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|value|`String`|Режим отображения представления|

## Returns

Метод ничего не возвращает.

# Examples

```js
linkView.setOpenMode('Dialog');
```
