Возвращает [режим отображения представления](../OpenMode/). 

# Syntax

```js
linkView.getOpenMode()
```

## Parameters

Нет

## Returns

Режим отображения представления. 

# Examples

```js
var openMode = linkView.getOpenMode();
```
