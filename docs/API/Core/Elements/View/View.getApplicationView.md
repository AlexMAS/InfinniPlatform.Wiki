Возвращает корневое представление, т.е. представление, у которого нет родительского элемента.

# Syntax

```js
view.getApplicationView()
```

## Parameters

Нет

## Returns

Корневое представление.

# Examples

```js
var rootView = view.getApplicationView();
```
