Устанавливает обработчик события выхода пользователя из системы.

# Syntax

```js
Session.onSignOut(callback);
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`callback`|`Function`|Обработчик события выхода пользователя из системы.|

# Returns

Нет
