Возвращает правило сортировки документов.

# Syntax

```js
DocumentDataSource.getOrder()
```

## Parameters

Нет

## Returns

Правило сортировки документов (см. [формат](../DocumentDataSource.setOrder/#order-format))

# Examples

```js
var order = DocumentDataSource.getOrder();
```
