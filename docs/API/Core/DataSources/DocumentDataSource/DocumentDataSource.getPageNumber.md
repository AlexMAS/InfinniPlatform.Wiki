Возвращает номер страницы.

# Syntax

```js
BaseDataSource.getPageNumber()
```

## Parameters

Нет

## Returns

Номер страницы. Целое число, начиная с 0.

# Examples

```js
var pageNumber = BaseDataSource.getPageNumber();
```
