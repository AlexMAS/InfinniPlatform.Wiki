Возвращает правило отображения документов.

# Syntax

```js
DocumentDataSource.getSelect()
```

## Parameters

Нет

## Returns

Правило отображения документов (см. [формат](../DocumentDataSource.setSelect/#select-format))

# Examples

```js
var selectFormat = DocumentDataSource.getSelect();
```
