Возвращает строку полнотекстового поиска.

# Syntax

```js
DocumentDataSource.getSearch()
```

## Parameters

Нет

## Returns

Строка полнотекстового поиска.

# Examples

```js
var searchStr = DocumentDataSource.getSearch();
```
