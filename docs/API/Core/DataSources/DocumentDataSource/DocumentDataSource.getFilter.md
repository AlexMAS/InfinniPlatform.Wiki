Возвращает правило фильтрации документов.

# Syntax

```js
DocumentDataSource.getFilter()
```

## Parameters

Нет

## Returns

Правило фильтрации документов (см. [формат](../DocumentDataSource.setFilter/#filter-format))

# Examples

```js
var filter = DocumentDataSource.getFilter();
```
