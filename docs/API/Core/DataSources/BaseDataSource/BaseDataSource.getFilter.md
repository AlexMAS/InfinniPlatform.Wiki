Возвращает фильтр элементов.

# Description

Фильтр элементов представляется массивом критериев фильтрации. 

# Syntax

```js
BaseDataSource.getFilter()
```

## Parameters

Нет

## Returns

Массив критериев фильтрации

# Examples

```js
var filter = BaseDataSource.getFilter();
```
