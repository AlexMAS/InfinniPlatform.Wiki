Возвращает строку полнотекстового поиска.

# Syntax

```js
BaseDataSource.getSearch()
```

## Parameters

Нет

## Returns

Строка полнотекстового поиска.

# Examples

```js
var searchStr = BaseDataSource.getSearch();
```
