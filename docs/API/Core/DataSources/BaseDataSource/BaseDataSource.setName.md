Устанавливает наименование источника данных.

# Syntax

```js
BaseDataSource.setName(value)
```

## Parameters

|Name|Description|
|----|-----------|
|value|Наименование источника данных|

## Returns

Метод ничего не возвращает.

# Examples

```js
BaseDataSource.setName('dataSource1');
```
