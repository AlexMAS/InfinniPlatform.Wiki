Возвращает родительское [представление](../../../Elements/View/).

# Syntax

```js
BaseDataSource.getView()
```

## Parameters

Нет

## Returns

Родительское [представление](../../../Elements/View/) источника данных.

# Examples

```js
var parentView = BaseDataSource.getView();
```
