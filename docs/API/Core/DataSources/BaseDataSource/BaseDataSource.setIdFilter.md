Устанавливает фильтр элементов по идентификатору.

# Syntax

```js
baseDataSource.setIdFilter(itemId)
```

## Parameters

|Name|Description|
|----|-----------|
|itemId|Идентификатор элемента|

## Returns

Метод ничего не возвращает.

# Examples

```js
baseDataSource.setIdFilter('dd4d1bb3-5571-453c-9686-d04d4d0f0316');
```
