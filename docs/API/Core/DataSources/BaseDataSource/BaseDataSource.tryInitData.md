Пытается обновить список элементов источника. Делает это только в случае, если до этого источник не был прогружен данными.

# Syntax

```js
BaseDataSource.tryInitData()
```

## Parameters

Нет

## Returns

Метод ничего не возвращает

# Examples

```js
BaseDataSource.tryInitData();
```
