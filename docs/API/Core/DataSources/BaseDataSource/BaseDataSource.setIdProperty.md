Устанавливает наименование свойства с идентификатором элемента.

# Syntax

```js
BaseDataSource.setIdProperty(value)
```

## Parameters

`value`

Наименование свойства с идентификатором элемента.

# Examples

```js
BaseDataSource.setIdProperty('Id');
```
