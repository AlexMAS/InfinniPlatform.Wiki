Определяет обязательное свойство [`dataProvider`](../../../DataProviders/). Наследники BaseDataSource обязаны переопределять этот метод!

# Syntax

```js
BaseDataSource.initDataProvider()
```

## Parameters

Нет

## Returns

Метод ничего не возвращает

# Examples

```js
BaseDataSource.initDataProvider();
```
