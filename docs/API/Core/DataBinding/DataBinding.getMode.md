Возвращает [направление обмена данными в привязке](../BindingMode/).

# Syntax

```js
dataBinding.getMode()
```

## Parameters

Нет

## Returns

[Направление обмена данными в привязке](../BindingMode/).

# Examples

```js
var bindingMode = dataBinding.getMode();
```
