Возвращает значение по умолчанию для элемента.

# Syntax

```js
dataBinding.getDefaultValue()
```

## Parameters

Нет

## Returns

Значение по умолчанию для элемента.

# Examples

```js
var defaultValue = dataBinding.getDefaultValue();
```
