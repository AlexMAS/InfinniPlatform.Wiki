Устанавливает [направление обмена данными в привязке](../BindingMode/).

# Syntax

```js
DataBinding.setMode(value)
```

## Parameters

`value`

[Направление обмена данными в привязке](../BindingMode/).

# Examples

```js
DataBinding.setMode(BindingMode.twoWay);
```
