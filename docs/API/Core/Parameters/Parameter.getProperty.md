Возвращает значение атрибута параметра.

# Syntax

```js
Parameter.getProperty(name);
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`name`|`String`|наименование параметра.|

# Returns

Значение атрибута параметра.

# Examples

```js
var firstName = Parameter.getProperty('first_name');
```
