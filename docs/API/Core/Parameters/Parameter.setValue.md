Устанавливает значение параметра.

# Syntax

```js
Parameter.setValue(value);
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`value`|&#42;|Значение параметра.|

# Returns

Нет

# Examples

```js
Parameter.setValue(33);
```
