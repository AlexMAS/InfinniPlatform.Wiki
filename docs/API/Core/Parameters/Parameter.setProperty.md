Устанавливает значение атрибута параметра.

# Syntax

```js
Parameter.setProperty(name, value);
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`name`|`String`|Имя атрибута.|
|`value`|&#42;|Значение атрибута.|

# Returns

Нет

# Examples

```js
Parameter.setProperty('qty', 33);
```
