Устанавливает наименование параметра.

# Syntax

```js
Parameter.setName(value);
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`value`|`String`|Наименование параметра.|

# Returns

Нет

# Examples

```js
Parameter.setName('parameter1');
```
