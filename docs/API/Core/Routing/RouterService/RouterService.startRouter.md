Запускает роутер.

# Description

Запускает роутер с описанными в [InfinniUI.config.Routes](../../Routes) маршрутами.

# Syntax

```js
RouterService.startRouter()
```

## Parameters

Нет.

# Examples

```js
RouterService.startRouter();
```
