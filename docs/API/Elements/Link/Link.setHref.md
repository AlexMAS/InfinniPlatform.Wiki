Устанавливает адрес документа, на который должна переводить ссылка.

# Syntax

```js
Link.setHref(value)
```

## Parameters

|Name|Type|Description|
|----|----------|---------|
|value|`String`|Адрес документа, на который должна переводить ссылка.|

## Returns

Нет.

# Examples

```js
link.setHref("http://docs.infinnity.ru/");
```
