Возвращает [`представление`](../../../Core/Elements/View), которое отображается внутри ViewPanel.

# Syntax

```js
viewPanel.getLayout()
```

## Parameters

Нет.

## Returns

[`Представление`](../../../Core/Elements/View), которое отображается внутри ViewPanel.

# Examples

```js
var layout = viewPanel.getLayout();
```
