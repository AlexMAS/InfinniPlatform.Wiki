# Extends

# Syntax

```js

```

## Parameters

|Name|Type|Default|Description|
|----|----|-------|-----------|
|`parent`|[`Element`](../../Core/Elements/Element)|-|Родительский элемент|
|`viewMode`|[`viewMode`](../../Core/viewMode/)|`common`|Режим отображения|
