Возвращает [функцию](../../../Core/Script/) шаблонизации содержимого кнопки.

# Syntax

```js
button.getContentTemplate()
```

## Parameters

Нет

## Returns

[Функция](../../../Core/Script/) шаблонизации содержимого кнопки.

# Examples

```js
var contentTemplate = button.getContentTemplate();
```
