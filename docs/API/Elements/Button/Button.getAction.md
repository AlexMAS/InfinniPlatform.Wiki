Возвращает [`действие`](../../../Core/Actions/) при нажатии на кнопку.

# Syntax

```js
button.getAction()
```

## Parameters

Нет

## Returns

[`Действие`](../../../Core/Actions/) при нажатии на кнопку.

# Examples

```js
var action = button.getAction();
```
