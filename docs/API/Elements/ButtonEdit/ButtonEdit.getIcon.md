Возвращает название иконки, отображаемой на кнопке.

# Syntax

```js
buttonEdit.getIcon()
```

## Parameters

Нет

## Returns

Название иконки.    
Полный список доступных названий и соответсвующих отображений: [Icons](http://fontawesome.io/icons/).

# Examples

```js
var icon = buttonEdit.getIcon();
```
