Устанавливает текст с информацией о предупреждении.

# Syntax

```js
EditorBase.setWarningText(value)
```

## Parameters

`value`

Текст с информацией об ошибке.

# Examples

```js
EditorBase.setWarningText('Maybe this value is too big');
```
