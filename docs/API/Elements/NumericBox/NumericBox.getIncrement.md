Возвращает приращение значения при вводе с помощью кнопок.

# Syntax

```js
NumericBox.getIncrement()
```

## Returns

Приращение значения при вводе с помощью кнопок

# Examples

```js
var maxValue = numericBox.getIncrement();
```
