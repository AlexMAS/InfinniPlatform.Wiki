Возвращает [ориентацию стека элементов](../StackPanelOrientation/).

# Syntax

```js
stackPanel.getOrientation()
```

## Parameters

Нет

## Returns

[Ориентация стека элементов](../StackPanelOrientation/).

# Examples

```js
var orientation = stackPanel.getOrientation();
```
