Устанавливает максимальное значение для ввода.

# Syntax

```js
DateTimePicker.setMaxValue(value)
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`value`|`Date`|Максимальное значение для ввода.|

# Examples

```js
dateTimePicker.setMaxValue(new Date());
```
