Устанавливает минимальное значение для ввода.

# Syntax

```js
DateTimePicker.setMinValue(value)
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`value`|`Date`|Минимальное значение для ввода.|

# Examples

```js
dateTimePicker.setMinValue(new Date());
```
