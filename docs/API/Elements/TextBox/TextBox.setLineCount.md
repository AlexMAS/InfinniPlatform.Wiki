Устанавливает максимально видимое количество строк.

# Syntax

```js
textbox.setLineCount(lineCount)
```

## Parameters

|Name|Type|Default|Description|
|----|----|-------|-----------|
|lineCount|`Integer`|-|Максимально видимое количество строк|

## Returns

Нет

# Examples

```js
textbox.setLineCount(10);
```
