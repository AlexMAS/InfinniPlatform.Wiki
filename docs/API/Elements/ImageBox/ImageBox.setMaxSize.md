Устанавливает максимальный размер данных в байтах.

# Syntax

```js
ImageBox.setMaxSize(value)
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`value`|`Number`|Максимальный размер данных в байтах|

## Returns

Нет.

# Examples

```js
imageBox.setMaxSize(1000000);
```
