Возвращает значение, определяющее, возможено ли изменение изображения.

# Syntax

```js
ImageBox.getReadOnly()
```

## Parameters

Нет

## Returns

Значение, определяющее, возможено ли изменение изображения.

# Examples

```js
var isReadOnly = imageBox.getReadOnly();
```
