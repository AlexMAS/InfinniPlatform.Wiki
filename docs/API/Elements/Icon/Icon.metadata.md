Метаданные типа [`Icon`](../).

# Properties

|Name|Type|Description|
|----|----|-----------|
|Size|[`TextStyle`](../../../Core/Style/TextStyle/)<sup>*</sup>|Размер иконки|

<sup>*</sup> в данном случае [`TextStyle`](../../../Core/Style/TextStyle/) определяет только размер шрифта.

# Examples

```json
{
    "Icon": {
        "Value": "star",
        "Size": "display1"
    }
}
```
