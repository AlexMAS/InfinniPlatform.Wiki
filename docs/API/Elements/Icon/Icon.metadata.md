Метаданные типа [`Icon`](../).

# Properties

|Name|Type|Default|Description|
|----|----|-------|-----------|
|Size|`String`| |Размер иконки, значения аналогичны свойству [`TextStyle`](../../../Core/Style/TextStyle/)|

# Examples

```json
{
    "Icon": {
        "Value": "star",
        "Size": "display1"
    }
}
```
