Устанавливает максимальный размер данных в байтах.

# Syntax

```js
FileBox.setMaxSize(value)
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|`value`|`Number`|Максимальный размер данных в байтах|

## Returns

Нет.

# Examples

```js
fileBox.setMaxSize(1000000);
```
