Возвращает максимальный размер данных в байтах.

# Syntax

```js
FileBox.getMaxSize()
```

## Parameters

Нет

## Returns

Максимальный размер данных в байтах.

# Examples

```js
var maxSize = fileBox.getMaxSize();
```
