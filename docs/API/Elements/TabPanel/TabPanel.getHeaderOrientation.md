Возвращает [ориентацию закладок](../TabHeaderOrientation/).

# Syntax

```js
tabPanel.getHeaderOrientation()
```

## Parameters

Нет

## Returns

[Ориентация закладок](../TabHeaderOrientation/).

# Examples

```js
var headerOrientation = tabPanel.getHeaderOrientation();
```
