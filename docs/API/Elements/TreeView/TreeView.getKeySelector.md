Возвращает [функцию](../../../Core/Script/) выборки из элемента коллекции идентификатора.

# Syntax

```js
TreeView.getKeySelector()
```

## Parameters

Нет

## Returns

[Функция](../../../Core/Script/) выборки из элемента коллекции идентификатора.

# Examples

```js
var selector = treeview.getKeySelector();
```
