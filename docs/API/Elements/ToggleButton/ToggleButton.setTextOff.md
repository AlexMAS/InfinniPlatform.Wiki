Устанавливает текст на выключенное состояние.

# Syntax

```js
toggleButton.setTextOff(value)
```

## Parameters

Name|Type|Description
----|----|-----------
`value`|`string`|текст на выключенное состояние.

## Returns

Нет.

# Examples

```js
toggleButton.setTextOff("OFF");
```
