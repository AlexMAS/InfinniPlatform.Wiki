Устанавливает количество страниц.

# Syntax

```js
DataNavigation.setPageCount(value)
```

## Parameters

Name|Type|Description
----|----|-----------
value|`Number`|Количество страниц.

## Returns

Нет.

# Examples

```js
dataNavigation.setPageCount(true);
```
