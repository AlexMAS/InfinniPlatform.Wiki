Устанавливает значение которое показывает готовы ли данные.

# Syntax

```js
DataNavigation.setIsDataReady(value)
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|value|`Boolean`|Готовность данных.|

## Returns

Нет.

# Examples

```js
dataNavigation.setIsDataReady(true);
```
