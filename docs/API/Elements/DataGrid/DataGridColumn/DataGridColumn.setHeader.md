Устанавливает заголовок колонки таблицы.

# Syntax

```js
DataGridColumn.setHeader(value);
```

## Parameters

|Name|Type|Description|
|----|----|-----------|
|value|`String`|Заголовок колонки таблицы.|

## Returns

Нет

# Examples

```js
column.setHeader("Id");
```
