Возвращает boolean-значение о том, есть ли шаблон ячейки заголовка.

# Syntax

```js
DataGridColumn.getIsHeaderTemplateEmpty();
```

## Parameters

Нет

## Returns

true или false.

# Examples

```js
var headerTemplate = DataGridColumn.getIsHeaderTemplateEmpty();
```
