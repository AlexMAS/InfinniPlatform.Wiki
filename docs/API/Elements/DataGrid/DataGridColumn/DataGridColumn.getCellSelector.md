Возвращает значение селектора ячейки.

# Syntax

```js
DataGridColumn.getCellSelector();
```

## Parameters

Нет

## Returns

Значение селектора ячейки.

# Examples

```js
var cellSelector = DataGridColumn.getCellSelector();
```
