Возвращает [коллекцию](../../../Core/Collection/) [колонок](../DataGridColumn/) таблицы.

# Syntax

```js
DataGrid.getColumns()
```

## Parameters

Нет

## Returns

[Коллекция](../../../Core/Collection/) [колонок](../DataGridColumn/).

# Examples

```js
var columns = DataGrid.getColumns();
```
