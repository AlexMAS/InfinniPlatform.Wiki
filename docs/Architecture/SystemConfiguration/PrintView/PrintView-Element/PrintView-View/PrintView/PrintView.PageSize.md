Размер страницы.

   

По умолчанию размер страницы печатного представления равен размеру бумажного листа [формата A4](https://en.wikipedia.org/wiki/Paper_size) (с портретной ориентацией).

       

#### Type

[[Size]]

   

#### Example

[[PrintView.PageSize.pdf]]

```
{
  "PageSize": {
    "Width": 210,
    "Height": 297,
    "SizeUnit": "Mm"
  }
}
```
